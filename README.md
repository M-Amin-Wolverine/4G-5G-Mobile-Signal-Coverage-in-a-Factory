# 4G-5G-Mobile-Signal-Coverage-in-a-Factory

This repository contains the dataset from measurements of 4G and 5G mobile signal coverage in the factory of [Tajmac-ZPS s.r.o.](https://www.tajmac-zps.cz/tajmac-group/), located in ZlÃ­n, Czech Republic. The measurment were conducted in  at various locations within the factory.
More details are available in the article "_Measurement and Analysis of 4G/5G Mobile Signal Coverage in a Heavy Industry Environment_".
